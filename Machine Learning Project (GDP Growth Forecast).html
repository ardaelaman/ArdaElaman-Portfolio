Forecasting U.S. GDP Growth: Machine Learning Methods vs AR Models

View Full Project

Introduction

This project analyzes the predictive power of machine learning models for forecasting quarterly U.S. GDP growth, comparing them with the traditional Autoregressive (AR(4)) model. It also examines the impact of key economic indicators on GDP growth using macroeconomic data from the Federal Reserve Economic Data (FRED), covering 1970Q1–2020Q1.

Data and Feature Selection

The dataset includes 21 economic indicators (inflation, private investment, unemployment rate, current account balance, etc.). Feature importance was ranked using Random Forest and Gradient Boosting, while PCA reduced dimensionality.

Machine Learning Models and Benchmarking

  • AR(4): baseline autoregressive model
  • Random Forest: ensemble of decision trees
  • Gradient Boosting: sequential boosting method

Training: 1970Q1–2012Q4 | Testing: 2013Q1–2020Q1 | Evaluation: MSE

Model Performance and Findings

  • Gradient Boosting: best test MSE (0.113)
  • Random Forest: 0.120
  • AR(4): worst performance (0.157)
  • Top drivers: private investment, labor productivity, inflation

Limitations and Future Work

  • Moderate prediction accuracy
  • Could integrate alternative datasets
  • Worth testing neural networks & deep learning
  • Geopolitical/trade data might improve results
  • Time-series cross-validation recommended

Conclusion

Machine learning models (RF, GB) significantly outperform the traditional AR(4) model in forecasting GDP growth. Gradient Boosting delivered the best accuracy, capturing non-linear dynamics. SHAP analysis highlighted investment and productivity as key growth drivers.
